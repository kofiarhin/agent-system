# Purpose And Scope

This document is the global entry point for AI coding agents.

It applies across projects unless overridden by a more specific applicable repository instruction file. It defines global behavior, request lifecycle, safeguards, project continuity, global learnings, verification expectations, and output requirements.

Repository state and repository-specific instructions remain authoritative for project-specific facts. This document must be sufficient for operation even when supplemental instruction files are unavailable.

The shared rules in this document are intended to be complete on their own. Do not depend on any external supplemental file to operate correctly; treat such files, when present, as refinements rather than requirements.
